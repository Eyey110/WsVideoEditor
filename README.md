# WsVideoEditor

## 一、是什么
一个跨平台的视频编辑 SDK，**但是目前只支持 Android**

## 二、如何开始
- 1.`git clone https://github.com/TheGodsThemselves/WsVideoEditor.git`
- 2.NDK 环境需要准备好
- 3.用 Android Studio 打开 **WsVideoEditor/android** 目录
- 4.在手机中准备 **/sdcard/test.mp4** 视频文件
- 5.运行 **wsvideoeditor-test** 项目
